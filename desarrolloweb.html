fase 1

definición y alcance del proyecto

Lo primero una vez hayas contratado el proyecto será establecer una reunión por videoconferencia con el Project Manager asignado a tu proyecto.

En esta primera fase definiremos junto a ti el público objetivo asi como toda la estructura de navegacion y funcionalidades de la web.

También analizaremos si tus textos e imágenes son los adecuados para el proyecto o requieren mejorarse.

fase 2

diseño

Crearemos un diseño premium y completamente personalizado de tu sitio web que posteriormente te enviaremos para que lo valides.

  • - Diseño de los frameworks para definir la estructura del site.

  • - Diseño de las páginas (Home, Servicios, About us, contacto, etc.)
    en versión web y móvil

fase 3

desarrollo

  • - Creación y desarrollo en HTML, javascript, y CSS de las páginas diseñadas
    previamente.

  • - Diseño de los frameworks para definir la estructura del site.

  • - Haremos un desarrollo completamente personalizado, profesional y sin utilizar ningún framework tipo de wordpress

  • - Implementación de formulario en la página de Contacto.

  • - Incluye el desarrollo de 5 páginas de contenido de hasta 1.000 palabras cada página.

*Los textos y las imágenes deben ser facilitados por el cliente.

fase 4

migración

Una vez finalizado el desarrollo web, se realizará la migración al entorno real y se realizará una optimización de velocidad del sitio web. De esta
forma, se finalizará el proyecto web.
